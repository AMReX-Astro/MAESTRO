∗ Get the Code

Maestro and the support libraries are distributed via git.

  1. Get MAESTRO from github: https://github.com/BoxLib-Codes/MAESTRO
    Note: if you previously cloned MAESTRO from our local repository, you can convert to the github version via:
    git remote set-url origin ssh://git@github.com/BoxLib-Codes/MAESTRO.git
  2. Subscribe to the maestro-help mailing list.
  3. Get BoxLib: instructions on the BoxLib Download page.
  4. Build the User's Guide in MAESTRO/Docs/ by typing make
  5. Following the instructions in the Getting Started chapter to run your first problem

∗ Keeping Up-to-date

The Maestro source is periodically updated. To update your source tree, simply type git pull in your MAESTRO/ directory. Note: you cannot push back to the public git repo.

ChangeLog:

It is also important to keep up to date with the BoxLib git repo.
